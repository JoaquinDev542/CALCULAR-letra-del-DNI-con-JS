# CALCULAR-letra-del-DNI-con-JS
En este repositorio, encontrarás una práctica hecha con JavaScript que nos permitirá saber que letra del DNI se tiene mediante sus números.

En estos archivos encontrarás una práctica para poder saber que letra tiene nuestro DNI sabiendo solo los números que lleva, de esta manera, practicaremos 
y reforzaremos temario utilizando el operador Switch, usando condicional tipi if-else, entre otras cosas.
